## 🚀 Installation

1. Download ZIP
2. WordPress → Appearance → Themes → Upload Theme
3. Activate
4. Settings → Reading → Static Page → Home
5. Done! Content automatically loads!

---

## ⚙️ Dashboard Management

### Custom Post Types
- **Services** — Add/Edit/Delete services with icons & tags
- **Testimonials** — Manage client reviews with ratings
- **Tools** — Update tech stack
- **Process** — Edit process steps with order

### Theme Customizer
- Hero Section — Badge, title, description, Upwork URL
- Profile Card — Name, title, stats, profile image
- Why Me Section — Big stat, mini stats
- Contact Info — Email, location, response time
- Footer — Copyright text
